## âš™ï¸ How It Works

1. Questions are stored inside `questions.json`.
2. Fetch API loads the JSON data.
3. Questions are shuffled randomly.
4. Timer (10 seconds) starts for each question.
5. After selection:
   - Options get disabled
   - Correct answer is highlighted
   - Score updates dynamically
6. Progress bar updates based on question index.
7. Final score is displayed at the end.
